# Plugin-AntirelogFork-1.16.5

Разработано студией @StarsDev - TG Инструкция по компиляции плагина для Minecraft через Maven

Установленная Java Development Kit (JDK) версии 16 или выше

Установленный Apache Maven

Установите исходный код нашего плагина

Соберите плагин через команду в терминале "mvn package"

Готовый плагин будет лежать в папке c:/users/ваше имя/IdeaProjects/Antirelog/Target

Плагин поддерживается на версии 1.16.5 paper minecraft (java)
